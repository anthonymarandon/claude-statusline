# Mise à jour de la statusline

Tu dois mettre à jour la statusline en exécutant le script de mise à jour.

## Ce que tu fais

1. Lance la commande suivante :

```bash
bash ~/.claude/update.sh
```

2. Lis la sortie du script et informe l'utilisateur du résultat :
   - Si la mise à jour a réussi : confirme la nouvelle version installée et rappelle de relancer Claude Code
   - Si une erreur s'est produite : explique le problème et propose une solution
   - Si la statusline n'est pas installée : indique la commande d'installation

## Ton

Direct et concis. Pas besoin de longues explications — l'utilisateur veut juste que ça se mette à jour.
